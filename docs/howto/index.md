# How-to guides

<!--8<-- "feedback.md"-->

This section provides a collection of step-by-step instructions intended to help you accomplish essential tasks related to the Domino REST API. It's organized into the following categories for easy reference:

## [Installation](install/index.md)

Guides for completing goals and tasks related to installation, configuration, and settings for Domino REST API.

## [Web](web/index.md)

Guides for configuring nginx as an HTTPS proxy for the Domino REST API.

!!! note

    Domino REST API is compatible with any type of proxy.

## [Identity Provider (IdP)](IdP/index.md)

Guides on how to use Domino as your Identity Provider (IdP) and how to configure external IdPs to be used with Domino.

## [Production](production/index.md)

Guides to help you deploy the Domino REST API reliably, securely, and at scale in production environments.

## [Database](database/index.md)

Guides for working with databases, including enabling and configuring for REST access.

## [Management console](management/index.md)

Guides to help you complete tasks and achieve goals related to the Domino REST API that require access to the Management console.

## [Code examples](codesamples/index.md)

Code samples to interact with Domino REST API.
