# Extensions used to open the Microsoft Office in OFBA
 
Here are the file formats often used by various applications within the Microsoft Office suite and Office 365 apps. Office Form-Based AuthenticationÂ can access these file types through the Domino REST API to open and modify office documents.

    ```
    csv=ms-excel:ofv|u|%s,ms-excel:ofe|u|%s
    ```
